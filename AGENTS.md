# MoveNow — Codex 작업 지침

## 프로젝트 개요
운동과 식단 기록을 기반으로 **다음 행동을 제안**하는 웹 애플리케이션.
단순 기록이 아닌 "기록 → 분석 → 행동" 흐름을 핵심으로 한다.

## 너의 역할
Claude가 수립한 계획을 코드로 구현한다.
아래 **현재 계획** 섹션을 반드시 먼저 읽고, 그 범위 안에서만 작업한다.

## 코딩 규칙
- 계획에 명시된 파일과 범위만 수정한다
- 기존 코드 패턴과 네이밍 컨벤션을 따른다
- 계획에 없는 기능은 추가하지 않는다
- 라이브러리 사용 시 `use context7`으로 최신 문서를 참조한다

## 기술 스택
> 확정 후 업데이트 예정

## 현재 계획
> Claude가 계획 수립 후 이 섹션을 업데이트함
> 작업 전 반드시 이 섹션을 확인할 것
