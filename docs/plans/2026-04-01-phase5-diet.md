## Step 1. Open Food Facts API 유틸 함수

`src/lib/openfoodfacts.ts` 생성:

```ts
const OFF_BASE = 'https://world.openfoodfacts.org'

export type FoodItem = {
  id: string
  name: string
  calories_per_100g: number
  protein_per_100g: number
  carbs_per_100g: number
  fat_per_100g: number
}
```

구현 포인트:

- `fetch`에 `{ next: { revalidate: 86400 } }` 옵션 → 하루 캐싱
- 응답 파싱 실패 / 영양정보 누락 항목은 필터링해서 제외
- 에러 시 빈 배열 반환 (throw 금지)

엔드포인트:

```
GET /cgi/search.pl?search_terms={query}&json=true&fields=product_name,nutriments&page_size=10
```

사용 필드:

- `product_name` → `name`
- `nutriments['energy-kcal_100g']` → `calories_per_100g`
- `nutriments['proteins_100g']` → `protein_per_100g`
- `nutriments['carbohydrates_100g']` → `carbs_per_100g`
- `nutriments['fat_100g']` → `fat_per_100g`

## Step 2. Diet Server Actions

`src/lib/actions/diet.ts` 생성:

**Zod 스키마:**

```ts
const DietSchema = z.object({
  date: z.string().regex(/^\d{4}-\d{2}-\d{2}$/),
  meal_time: z.enum(['breakfast', 'lunch', 'dinner', 'snack']),
  food_name: z.string().trim().min(1).max(100),
  calories: z.coerce.number().int().min(0).max(5000),
  protein_g: z.coerce.number().min(0).max(500),
  carbs_g: z.coerce.number().min(0).max(500),
  fat_g: z.coerce.number().min(0).max(500),
  amount_g: z.coerce.number().min(1).max(5000).optional(),
})
```

**Server Actions:**

```ts
// 식단 기록 추가
export async function createDietAction(
  prevState: CreateDietState,
  formData: FormData,
): Promise<CreateDietState>

// 식단 기록 삭제
export async function deleteDietAction(id: string): Promise<{ error: string | null }>
```

`createDietAction` 처리 순서:

1. Zod 파싱
2. `amount_g`가 있으면 영양소를 비율로 재계산 (100g 기준 → 실제 섭취량)
3. `createDietLog()` 호출
4. `revalidatePath('/diet')`

## Step 5. Diet 컴포넌트

**`src/components/diet/DietForm.tsx`** — `'use client'`:

- 음식 검색 입력 + 검색 버튼
- 검색 결과 드롭다운 → 선택 시 영양정보 자동 입력
- 선택 후 `amount_g` 입력 시 칼로리·영양소 실시간 재계산 표시
- 필드: 날짜, 식사 시간, 음식명, 칼로리, 단백질, 탄수화물, 지방, 섭취량(g, 선택)
- 저장 시 `createDietAction` 호출

**`src/components/diet/DietList.tsx`** — Server Component:

- 당일 식단 목록 — 식사 시간별 그룹 표시 (아침 / 점심 / 저녁 / 간식)
- 하단에 오늘 합계: 총 칼로리 / 단백질 / 탄수화물 / 지방

**`src/components/diet/DietCard.tsx`** — Server Component:

- 단일 식단 기록 카드
- 음식명 / 칼로리 / 단백질·탄수화물·지방 / 식사 시간 Badge
- 삭제 버튼 → `deleteDietAction` inline Server Action
