# MoveNow

**Move now. No excuses.**
지금 움직여. 핑계는 없다.

---

## 🚀 소개

MoveNow는 운동과 식단을 단순히 기록하는 것을 넘어서,
**개인의 신체 정보와 행동 데이터를 기반으로 방향을 제시하는 웹 애플리케이션**입니다.

운동을 얼마나 했는지보다 중요한 것은
지금 나에게 무엇이 부족한지 아는 것입니다.

> MoveNow는 기록을 통해 “다음 행동”을 알려주는 도구입니다.

---

## 🎯 목표

* 운동과 식단을 **자유롭게 기록할 수 있는 환경 제공**
* 개인의 신체 정보 기반으로 **맞춤형 기준 생성**
* 현재 상태를 분석하여 **부족한 행동을 제안**
* 계획이 아닌 **실행을 유도하는 시스템 구축**

---

## 🧠 핵심 개념

### 1. 자유로운 기록

* 근력 / 유산소 구분 없이 모든 운동 기록 가능
* 운동 형태에 제한 없는 유연한 구조

### 2. 개인 기준 생성

* 체중, 목표, 활동량 등의 정보를 기반으로
* 개인에게 맞는 기준 자동 설정

### 3. 데이터 기반 피드백

* “얼마나 했는가”가 아니라
  👉 “무엇이 부족한가”에 집중

예시:

* 단백질 섭취 부족 → 추가 섭취 필요
* 운동량 부족 → 추가 활동 필요

### 4. 행동 유도 시스템

* 단순 통계가 아닌
  👉 **다음 행동을 제시하는 구조**

---

## 📦 주요 기능 (예정)

### 🏋️ 운동 기록

* 운동 종류 제한 없음 (근력 / 유산소 포함)
* 운동 시간, 강도, 볼륨 등 자유 기록

### 🍽 식단 기록

* 음식 입력 및 영양 정보 기록
* 단백질 / 탄수화물 / 지방 추적

### 📊 상태 분석

* 현재 활동량 vs 기준 비교
* 영양 섭취 상태 분석

### ⚡ 행동 제안

* “오늘 단백질 부족”
* “운동량 부족”
* “추가 활동 필요”

👉 사용자가 다음 행동을 바로 할 수 있도록 안내

---

## 🧭 방향성

MoveNow는 단순 기록 앱이 아닙니다.

* 기록 → 분석 → 행동

이 흐름을 통해
사용자가 더 나은 선택을 할 수 있도록 돕습니다.

---

## 🧩 구조 (개념)

```
운동 데이터 + 식단 데이터
        ↓
개인 기준 (신체 정보 기반)
        ↓
비교 / 분석
        ↓
행동 제안
```
